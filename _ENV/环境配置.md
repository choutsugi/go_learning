## 一、Go下载

下载地址：https://studygolang.com/dl

查看版本：go version 

## 二、环境变量配置

```bash
$ go env -w GO111MODULE=auto
```



设置GOPATH路径（工作区）： go env -w GOPATH=F:\Workspace\Go

配置GoMOD代理（七牛云）：

```powershell
$ go env -w GO111MODULE=on
$ go env -w GOPROXY=https://goproxy.cn
```
