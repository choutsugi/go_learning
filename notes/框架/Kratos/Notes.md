## 一、安装

go install安装：

```bash
go install github.com/go-kratos/kratos/cmd/kratos/v2@latest
```

源码编译安装：

```bash
git clone https://github.com/go-kratos/kratos
cd kratos
make install
```

二、创建项目

```
kratos new helloworld
```
