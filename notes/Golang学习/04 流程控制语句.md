## 递增递减语句

Go语言只有后置自增和后置自减，没有前置自增和前置自减。

## 条件语句

示例：

```go
func main() {
	a:=0

	if a == 0 {
		fmt.Println("a 等于 0")
	} else if a == 1 {
		fmt.Println("a 等于 1")
	} else {
		fmt.Println("a 不等于 0 和 1")
	}
}
```

## 选择语句

示例：

```go
func main() {
	a:=0

	switch a {
	case 0:
		fmt.Println("a 等于 0")
	case 1:
		fmt.Println("a 等于 1")
		//fallthrough	// 直接进入下一个case，不执行break。
	default:
		fmt.Println("a 不等于 0 和 1")
	}
}
```

> case默认带有break，要想继续执行下一个case可使用fallthrough。

## 循环语句

**死循环：**

```go
func main() {
	a:=0

	for {
		a++
		if a>10 {
			break
		}
		fmt.Println(a)
	}
}
```

> 通过break跳出。

**for循环_方式1：**

```go
func main() {
	for a := 0; a < 10; a++ {
		fmt.Println(a+1)
	}
}
```

**for循环_方式2：**

```go
func main() {
	a:=0
	for a < 10 {
		a++
		fmt.Println(a)
	}
}
```

## 跳转语句

- break、continue：相互配合可用于多次循环嵌套的跳出。

  ```go
  func main() {
  	a:=0
  	for a < 10 {
  		a++
  		if a==5 {
  			continue	// 跳过本次循环
  		}
  		fmt.Println(a)
  	}
  }
  ```

- goto：调整执行位置。

  ```go
  func main() {
  	a:=0
  	A:
  		for a < 10 {
  			a++
  			if a==5 {
  				break A
  				goto B
  			}
  			fmt.Println("in label A: ", a)
  		}
  	B:
  		if a != 10 {
  			fmt.Println("in label B: ", a)
  			goto A
  		}
  }
  ```
