## 接口定义与使用

```go
type Animal interface {
	Eat()
	Run()
}

type Cat struct {
	Name string
	Sex bool
}

type Dog struct {
	Name string
}

func (dog Dog)Run(){
	fmt.Println(dog.Name, "Running..")
}

func (dog Dog)Eat(){
	fmt.Println(dog.Name, "Eating..")
}

func (cat Cat)Run(){
	fmt.Println(cat.Name, "Running..")
}

func (cat Cat)Eat(){
	fmt.Println(cat.Name, "Eating..")
}

func main() {
	var animal Animal
	animal = Cat{
		Name: "Tom",
		Sex:  true,
	}

	animal.Eat()
	animal.Run()
}
```

## 接口企业应用

先声明全局接口变量，在调用某个函数进行接口的实现后才能调用其方法。

```go
type Animal interface {
	Eat()
	Run()
}

type Cat struct {
	Name string
	Sex bool
}

type Dog struct {
	Name string
}

func (dog Dog)Run(){
	fmt.Println(dog.Name, "Running..")
}

func (dog Dog)Eat(){
	fmt.Println(dog.Name, "Eating..")
}

func (cat Cat)Run(){
	fmt.Println(cat.Name, "Running..")
}

func (cat Cat)Eat(){
	fmt.Println(cat.Name, "Eating..")
}

func myFunc(animal Animal){
	L = animal	//实现接口
}

var L Animal

func main() {
	tom := Cat{
		Name: "Tom",
		Sex:  true,
	}

	myFunc(tom)
	L.Run()
}
```
