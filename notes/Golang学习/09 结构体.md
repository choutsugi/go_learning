## 结构体定义

```go
type Ohiua struct {
	Name string
	Age int
	Sex bool
	Hobby []string
}
```

## 结构体变量声明

**方式一**

```go
var ohiua Ohiua
ohiua.Name = "Ohiua"
ohiua.Age = 24
ohiua.Sex = true
ohiua.Hobby = []string{"WatchMountain", "ListenSea"}
fmt.Println(ohiua)
```

**方式二**

```go
ohiua := Ohiua{
    Name:  "Ohiua",
    Age:   24,
    Sex:   true,
    Hobby: []string{"WatchMountain", "ListenSea"},
}
fmt.Println(ohiua)
```

**方式三**：new获得的变量为指针类型。

```go
ohiua := new(Ohiua)
ohiua.Name = "Ohiua"
ohiua.Age = 24
ohiua.Sex = true
ohiua.Hobby = []string{"WatchMountain", "ListenSea"}
fmt.Println(*ohiua)
```

## 结构体绑定函数

- 使得该函数成为结构体的成员函数。

```go
func(ohiua *Ohiua) Song(name string)(resStr string){
	resStr = name + "：我们一起学猫叫 一起喵喵喵喵喵"
	return
}

func main() {

	ohiua := Ohiua{
		Name:  "Ohiua",
		Age:   24,
		Sex:   true,
		Hobby: []string{"WatchMountain", "ListenSea"},
	}

	fmt.Println(ohiua.Song(ohiua.Name))
}
```

## 结构体嵌套

```go
type Home struct {
	City string
}

type Ohiua struct {
	Name string
	Age int
	Sex bool
	Hobby []string
	Homes Home
}

func main() {

	ohiua := Ohiua{
		Name:  "Ohiua",
		Age:   24,
		Sex:   true,
		Hobby: []string{"WatchMountain", "ListenSea"},
		Homes: Home{
			"DuanZhou",
		},
	}

	fmt.Println(ohiua.Homes.City)
}
```
