## 指针

```go
func main() {
	var a int
	var b *int

	a = 123
	b = &a
	fmt.Printf("address of a = %d, address of b = %d\n", &a, b)
	fmt.Printf("a = %d, b = %d\n", a, *b)

	*b = 456
	fmt.Printf("address of a = %d, address of b = %d\n", &a, b)
	fmt.Printf("a = %d, b = %d\n", a, *b)
}

/*
	输出：
        address of a = 824633762008, address of b = 824633762008
        a = 123, b = 123
        address of a = 824633762008, address of b = 824633762008
        a = 456, b = 456
*/
```

> b指向a的地址：一荣俱荣，一损俱损。

## 数组指针和指针数组

**数组指针：**数组的指针。

```go
// 数组指针：数组的指针
var arr [5]string
arr = [5]string {"1","2","3","4","5"}

var arrPtr *[5]string
arrPtr = &arr

fmt.Println(arr, arrPtr)
fmt.Println("arrPtr[0] = ", arrPtr[0])
fmt.Println("*arrPtr[0] = ", (*arrPtr)[0])
```

**指针数组：**存放指针的数组。

```go
// 指针数组：存放指针的数组。
var arrp [5]*string		//可存放5个string指针的数组。
var str1 string = "str1"
var str2 string = "str2"
var str3 string = "str3"
var str4 string = "str4"
var str5 string = "str5"

arrp = [5]*string{&str1,&str2,&str3,&str4,&str5}
fmt.Println(arrp)
for i, v := range arrp {
    *v = *v + strconv.Itoa(i+1)
    fmt.Println(*v)
}
```

## 通过指针传参

```go
func main() {
	str := "ohiua"
	changeString(&str)
	fmt.Println(str)
}

func changeString(str *string){
	*str = "hi ohiua"
}
```
